# AdvancedCalculator

A functional calculator made in java language.

It has a menu contains operations to do.

Accepts int and double values.

1- Addition <br>
2- Subtraction <br>
3- Multiplication <br>
4- Division <br>
5- Exponentiation <br>
6- Factorial <br>
7- Modulo <br>
8- Rectangle area and perimeter <br>
9- Exit
